- 설정 가능한 인게임 통계 오버레이: 진행도, 정확도, X-정확도, BPM, 타일 BPM, 타이밍 스케일, 판정, 콤보, 시도 횟수, FPS
- 커스터마이즈 가능한 키 뷰어: 키 레인, 고스트 키, 손/발 프리셋, 키별 및 전체 입력 횟수
- 키 리미터 및 채터링 차단
- 게임 전체 커스텀 폰트: 게임 자체 텍스트(메뉴, HUD, 레벨 제목)와 모드 UI의 글꼴 교체 (`.ttf`/`.otf` 파일을 `…/Bismuth/Fonts/`에 넣고 설정에서 선택 가능)
- 게임 UI 편집기: 개별 게임 HUD 요소를 이동·크기 조절·숨기기·굵기 변경 (사망 %, 축하 문구, 카운트다운, 판정선, 자동 플레이 텍스트 등)
- UI 숨기기 토글 (HUD, 판정선, 자동 플레이 텍스트/아이콘, 실패 방지 아이콘, 난이도 아이콘, 판정 텍스트, 곡 제목, 알파/베타 빌드 텍스트)
- 성능 최적화 및 편의 기능: 인게임 로그, 원클릭 강제 새로고침

테스트 환경:

- macOS 26.5, ADOFAI v3.2.0
- Windows 11, ADOFAI v3.2.0
- Arch Linux, 커널 7.0, ADOFAI v3.1.2

## 설치

1. 인게임 UMM 메뉴에서 Bismuth가 활성화되어 있는지 확인하세요.
2. [Releases](https://github.com/QuartzTeam/Bismuth/releases)에서 최신 `Bismuth.zip`을 다운로드하세요.
3. `…/A Dance of Fire and Ice/Mods/`에 압축을 해제하세요.
4. 게임을 실행하고 모드 메뉴에서 **Bismuth**를 켜세요.

### MelonLoader를 사용하는 경우

*수동 설치가 번거롭다면, 제 [설치 프로그램](https://github.com/sbrothers7/UMMInstall) 또는 [modlist.org 앱](https://github.com/modlist-org/modlist_org_app)을 사용해 보세요.*

1. UMMCompat 플러그인을 설치하세요.
2. `…/A Dance of Fire and Ice/UMMMods/`에 압축을 해제하세요.

개발자 참고 문서는 [REFERENCE.md](REFERENCE.md)를 확인하세요.
